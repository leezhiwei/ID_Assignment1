Images of KDE Desktop vs other OSes

Linux has a variety of Desktop Environments, from a simple one such as LXDE, to a more fully featured one, like KDE. As said previously, Linux is very customisable and can be used in many different applications, like Servers and Desktop machines. So, you would need to add the Desktop Environment package, if you have a base Linux install, or installed it automatically, in the case of a Distribution like Ubuntu, it is preinstalled and ready for usage.
