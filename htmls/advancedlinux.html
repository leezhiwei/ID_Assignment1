Advanced Linux Features

Linux is very advanced, and can be used to do many different things from compiling the kernel from scratch, to setting scheduled task in order to do some server maintainence. On the note of scheduling task, I will teach you how to set up a crontab, which is basically a script or command that you run regularly.

Below is the command for crontab and the syntax of it.

crontab {-l,e}

Basically -e is to edit the crontab file and -l is to list the contents of the crontab file.

Content of the crontab file.

The syntax of crontab content is shown below.

{minute} {hour} {day of month} {month of year} {day of week} {command or script to execute}

Take note that the time, eg minutes hour day can be replaced with * to repeat. Example of command or script is

touch /var/log/log.log

Installing packages on Linux

Every program on most Linux distributions are distributed as a package, so it is very important to know how to install a package though a command line. It also varies throughout which version of Linux you are using, regardless if it is Arch Linux, Ubuntu (Debian), or even Alpine Linux. I will try and cover it here. The order of installing packages is to, update the local repository, update local packages (if necessary), then install package.

For Debian-based distributions, the commands are shown below in sequence of update repository, upgrade packages, and finally installing the package itself. Additionally, last command is to search for packages.

sudo apt update

sudo apt upgrade

sudo apt install {package name}

sudo apt search {search term}

For Arch-based distributions, the commands are shown below in sequence of update repository, upgrade packages, and finally installing the package itself. Additionally, last command is to search for packages.

sudo pacman -Syy (case-sensitive)

sudo pacman -Syyu (case-sensitive)

sudo pacman -S {package name} (case-sensitive)

sudo pacman -Ss {search term}

For Alpine-based distributions, the commands are shown below in sequence of update repository, upgrade packages, and finally installing the package itself. Additionally, last command is to search for packages.

sudo apk update

sudo apk upgrade

sudo apk install {package name}

sudo apk search {search term}
