What is Linux?

Linux is a free and open-source operating system. Meaning that the source code for Linux is freely accessible by anyone and can be modified for any uses.
Basically speaking, Linux is a kernel and the other aspects of the operating systems like the graphics, is handled by the X Windows Systems, are handled by other software packages.
